Description:
	This program uses SDL to animate an image.
	The program cycles through the same animation
	three times.
	
Usage:
	A Makefile is provided, build an executable "run"
	with the "make" command. Program is then executed using
	"./run". Pressing F4 at the beginning of animation will
	generate and store frames in the provided frames directory.
	Press ESC to exit the animation window.
